# kafka2webex
This repository is an example to integrate both Apache Kafka and Cisco Webex Teams.
<br><br>

# Installation of Apache Kafka
<br><br>

# How to run
Edit webex token and room id at webex.properties
```
[DEFAULT]
WEBEX_TEAMS_ACCESS_TOKEN = CHANGE_ME
WEBEX_TEAMS_ROOM_ID = CHANGE_ME
```

Then, run...
```python
python sample.py
```
